Задача №27 (вариант 80)
==================================================

80) (А. Кабанов) Учёный решил провести кластеризацию  некоторого  множества
    звёзд по их расположению на карте звёздного неба… Средним расстоянием S
    назовём среднее арифметическое расстояние между всеми парами  различных
    точек в кластере. Для каждого файла  определите  среднее  расстояние  в
    каждом кластере, затем вычислите два числа: Smin – минимальное  среднее
    расстояние кластера, и Smax – максимальное среднее расстояние кластера.
    Исходные данные находятся в файлах 27-80a.txt и 27-80b.txt.
    В ответе запишите четыре числа: в первой строке сначала целую часть
    абсолютного значения произведения Smin × 100 000, затем целую часть
    абсолютного значения произведения Smax × 100 000 для файла А, во второй
    строке – аналогичные данные для файла Б.
