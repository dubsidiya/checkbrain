Задача №18 (вариант 68)
==================================================

68)
68) (В.Н. Шубинкин) Исходные данные для Робота записаны в файле 18-1.xls в
    виде электронной таблицы прямоугольной формы. Робот может двигаться
    только вверх и вправо. С каждой клетки Робот забирает наибольшее
    количество контейнеров вместимостью 8 монет каждый, полностью
    заполненных монетами.  Определите максимальную и минимальную денежную
    сумму, которую может собрать Робот, пройдя из левой НИЖНЕЙ клетки в
    правую ВЕРХНЮЮ. В ответе укажите два числа – сначала максимальную
    сумму, затем минимальную.
