Задача №18 (вариант 67)
==================================================

67)
67) (В.Н. Шубинкин) Исходные данные для Робота записаны в файле 18-1.xls в
    виде электронной таблицы прямоугольной формы. Робот может двигаться
    только вниз и вправо. С каждой клетки Робот забирает наибольшее
    количество контейнеров вместимостью 8 монет каждый, полностью
    заполненных монетами. Определите максимальную и минимальную денежную
    сумму, которую может собрать Робот, пройдя из левой верхней клетки в
    правую нижнюю. В ответе укажите два числа – сначала максимальную сумму,
    затем минимальную.
