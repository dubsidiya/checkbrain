187)
187) (Е. Джобс) Алгоритм вычисления функции F(n), где n – натуральное
    число, задан следующими соотношениями:
         F(n) = n, если n ≥ 2022,
         F(n) = 7 + F(n + 5), если n < 2022.
    Чему равно значение выражения F(45) – F(49)?
