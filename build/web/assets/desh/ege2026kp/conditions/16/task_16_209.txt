209) (А. Минак) Алгоритм вычисления значения функции F(n), где n –
    натуральное число, задан следующими соотношениями:
         F(n) = 6, если n = 1,
         F(n) = 3n + 2 + F(n – 1), если n > 1.
    Чему равно значение выражения F(2024) – F(2020)?
