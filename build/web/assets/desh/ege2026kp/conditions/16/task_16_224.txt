224) (Демо-2025) Алгоритм вычисления значения функции F(n), где n –
    натуральное число, задан следующими соотношениями:
         F(n) = 1 при n = 1;
         F(n) = (n – 1)·F(n – 1), если n > 1.
    Чему равно значение (F(2024) + 2·F(2023)) / F(2022)?
