Задача №16 (вариант 129)
==================================================

129)
129) (А. Богданов) Алгоритм вычисления функции F(n), где n – целое число, задан следующими
соотношениями:
F(n) = n, если n < 2
F(n) = F(n / 2) + 1, если n ≥ 2 и число n чётное,
F(n) = F(3n + 1) + 1, если n ≥ 2 и число n нечётное.
Определите количество значений n на отрезке [1;100000], для которых F(n) равно 16.
