210) *(А. Минак) Алгоритм вычисления значения функции F(n), где n –
    натуральное число, задан следующими соотношениями:
         F(1) = F(2) = 1,
         F(n) = 3·F(n – 2) + F(n – 1), если n > 2.
    Чему равно значение выражения F(20000024) / F(20000020)? В ответе
    запишите целую часть результата.
