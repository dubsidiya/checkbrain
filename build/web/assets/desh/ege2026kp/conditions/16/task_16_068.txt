Задача №16 (вариант 68)
==================================================

68)
68) Алгоритм вычисления функций F(n), где n – натуральное число, задан
    следующими соотношениями:
         F(n) = n + 1 при n < 3,
         F(n) = F(n – 2) + n – 2, когда n ≥ 3 и четно,
         F(n) = F(n + 2) + n + 2, когда n ≥ 3 и нечетно.

    Сколько существует чисел n, для которых значение F(n) определено и
    будет пятизначным?
