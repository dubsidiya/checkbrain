==================================================

232)
232) *(О. Лысенков) Алгоритм вычисления значения функции F(n), где n –
    натуральное число, задан следующими соотношениями:
         F(n) = n + 1, если n ≤ 3;
         F(n) = F(n – 3) + n – 15, если n > 3 и n кратно 3;
         F(n) = F(n + 3) + 2n, если n > 3 и n не кратно 3.
