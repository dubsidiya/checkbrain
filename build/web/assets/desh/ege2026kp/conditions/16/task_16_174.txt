Задача №16 (вариант 174)
==================================================

174)
174) (А. Богданов) Алгоритм вычисления функции F(n), где n – натуральное число, задан
следующими соотношениями:
F(n) = n, если n ≥ 2020;
F(n) = n + 2 + F(n+3), если n < 2020.
Определите значение выражения F(2012) – F(2023).
