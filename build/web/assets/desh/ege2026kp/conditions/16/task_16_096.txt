Задача №16 (вариант 96)
==================================================

96)
96) (А. Богданов) Алгоритмы вычисления функций F(n) и G(n) заданы
    следующими соотношениями (здесь // – операция деления нацело, % –
    остаток от деления):
        F(n) = n, при n < 10,
        F(n) = n % 10 + F(n // 10), при n ≥ 10.
        G(n) = n, при n < 10,
        G(n) = G(F(n)), при n ≥ 10,

    Чему равна сумма значений функции G(n) для всех двузначных n?
