Задача №16 (вариант 194)
==================================================

194)
194) Алгоритм вычисления функции F(n), где n – натуральное число, задан
    следующими соотношениями:
         F(n) = n, если n ≥ 1900,
         F(n) = n · F(n + 1), если n < 1900 и n не делится на 3;
         F(n) = n · F(n + 2) / 3, если n < 1900 и n делится на 3.
    Чему равно значение выражения F(1875) / F(1880)?
