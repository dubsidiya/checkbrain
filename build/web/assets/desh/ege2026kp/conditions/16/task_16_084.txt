Задача №16 (вариант 84)
==================================================

84)
84) (А. Богданов) Алгоритм вычисления функции F(n), где n – целое число,
    задан следующими соотношениями:
         F(n) = n, при n < 2,
         F(n) = F(n / 2) + 1, когда n ( 2 и чётное,
         F(n) = F(3n + 1) + 1 , когда n ( 2 и нечётное.

    Назовите количество значений n на отрезке [1;100], для которых F(n)
    определено и больше 100.
