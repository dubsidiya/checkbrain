Задача №16 (вариант 125)
==================================================

125) (Е. Джобс) Алгоритм вычисления функции F(n), где n – целое число,
    задан следующими соотношениями:
         F(n) = 1, если n < 3
         F(n) = F(n – 1) – F(n – 2), если n > 2 и сумма цифр числа n чётная,
         F(n) = F(n – 1) + F(n // 2), если n > 2 и сумма цифр числа n
        нечётная.
    Здесь символы // означают деление нацело. Вычислите значение F(100).
