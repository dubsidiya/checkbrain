57) Алгоритм вычисления функции F(n) задан следующими соотношениями:
F(n) = n · n + 2 · n + 1, при n > 25
F(n) = 2 · F(n+1) + F(n+3), при чётных n  25
F(n) = F(n+2) + 3 · F(n+5), при нечётных n  25
Определите количество натуральных значений n из отрезка [1; 1000], при которых значение F(n)
не содержит цифру 0.
