136) (К. Багдасарян) Алгоритм вычисления функции F(n), где n – натуральное
    число, задан следующими соотношениями:
         F(n) = 1, если n < 3
         F(n) = F(n – 1) + F(n – 2), если n > 2.
    Чему равно значение выражения (F(1006) – F(1004)) / F(1005)?
