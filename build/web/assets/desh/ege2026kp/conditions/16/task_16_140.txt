==================================================

140)
140) (А. Куканова) Алгоритм вычисления функции F(n), где n – натуральное
    число, задан следующими соотношениями:
         F(n) = n – 10000, если n > 10000,
         F(n) = F(n + 1) + F(n + 2), если 1 ≤  n ≤ 10000.
    Чему равно значение выражения F(12345)·(F(10) − F(12)) / F(11) +
    F(10101)?
