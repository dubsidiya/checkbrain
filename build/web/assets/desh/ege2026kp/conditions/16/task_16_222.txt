222) (Д. Муфаззалов) Обозначим через a // b целую часть от частного при
    делении числа a на число b. Алгоритм вычисления значения функции F(n),
    где n – натуральное, задан следующими соотношениями:
         F(n) = 1 при n ≤ 2025;
         F(n) = F( (n + 2024) // 2025) + 1, если n > 2025.
    Чему равно значение F(20252025)?
