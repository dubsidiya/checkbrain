Задача №16 (вариант 77)
==================================================

77)
77) Алгоритм вычисления функции F(n), где n – целое число, задан следующими
    соотношениями:
         F(n) = n, при n ( 1,
         F(n) = n + F(n / 3), когда n > 1 и делится на 3,
         F(n) = n + F(n + 3) , когда n > 1 и не делится на 3.

    Назовите минимальное значение n, для которого F(n) определено и больше
    100.
