Задача №16 (вариант 191)
==================================================

191)
191) (PRO100-ЕГЭ) Алгоритм вычисления функций F(n) и G(n), где n – целое число, задан следующими
соотношениями:
F(n) = G(n – 1),
G(n) = n, если n < 10,
G(n) = G(n – 2) + 1, если n ≥ 10.
Определите количество значений n на отрезке [1, 100], для которых значение функции F(n) будет
полным квадратом некоторого натурального числа.
