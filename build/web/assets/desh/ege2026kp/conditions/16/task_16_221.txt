221) (Д. Муфаззалов) Обозначим через a // b целую часть от частного при
    делении числа a на число b. Алгоритм вычисления значения функции F(n),
    где n – натуральное, задан следующими соотношениями:
         F(n) = 1 при n < 3;
         F(n) = F( (n + 1) // 2) + 1, если n > 2.
    Чему равно значение F(21 + 22 + 23 + … + 22025)?
