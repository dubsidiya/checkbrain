Задача №7 (вариант 79)
==================================================

79) В информационной системе хранятся изображения размером 2048 × 1600
    пикселей. При кодировании используется алгоритм сжатия изображений,
    позволяющий уменьшить размер памяти для хранения одного изображения в
    среднем в 8 раз по сравнению с независимым кодированием каждого
    пикселя. Каждое изображение дополняется служебной информацией, которая
    занимает 64 Кбайт. Для хранения 32 изображений выделено 12 Мбайт
    памяти. Какое максимальное количество цветов можно использовать в
    палитре каждого изображения?
