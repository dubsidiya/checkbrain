Задача №7 (вариант 96)
==================================================

96) (А. Богданов) Изображение размером 265х2084 пикселей сохраняется в
    памяти компьютера. Для его хранения выделяется не более 400Кбайт без
    учёта заголовка файла. Все пиксели кодируются одинаковым количеством
    бит и записываются в файл один за другим. Какое максимальное количество
    цветов может использоваться для хранения такого изображения?
