Задача №3 (вариант 95)
==================================================

95) (И. Женецкий) В файле 3-95.xls приведён фрагмент базы данных «Фриланс»
    о заработке исполнителей фриланс-биржи. База данных состоит из трёх
    таблиц. Таблица «Заказы» содержит записи о выполненных заказах за
    определённый период. Поле Код услуги содержит идентификатор услуги,
    поле Код исполнителя содержит код исполнителя заказа, а поле Стоимость
    содержит стоимость заказа в рублях. Таблица Услуги содержит информацию
    об услугах. Таблица Исполнители содержит информацию об исполнителях
    заказов. На рисунке приведена схема указанной базы данных.
                                    [pic]
    Используя информацию из приведённой базы данных, определите код
    исполнителя старше 30 лет с гражданством Китая, который заработал
    больше всего при оказании услуги Python-программист.
