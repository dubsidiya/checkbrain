Задача №14 (вариант 436)
==================================================

436) (ЕГЭ-2023) Операнды арифметического выражения записаны в системе
    счисления с основанием 22:

                       18x8995722 + 80х3322 + 521x622

    В записи чисел переменной x обозначена неизвестная цифра из алфавита 22-
    ричной системы счисления. Определите наименьшее значение x, при котором
    значение данного арифметического выражения кратно 21. Для найденного
    значения x вычислите частное от деления значения арифметического
    выражения на 21 и укажите его в ответе в десятичной системе счисления.
    Основание системы счисления указывать не нужно.
