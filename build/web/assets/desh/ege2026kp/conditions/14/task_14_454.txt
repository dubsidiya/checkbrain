Задача №14 (вариант 454)
==================================================

454) (М. Ишимов) Операнды арифметического выражения записаны в системе
    счисления с основанием 15.
          9xAB3x6E15 + 6B53x15 + 935DxBAx15 + E7Dxx6615 + DxE51415
    В записи чисел переменной x обозначена неизвестная цифра из алфавита 15-
    ричной системы счисления. Определите наименьшее значение x, при котором
    значение данного арифметического выражения кратно 14. Для найденного x
    вычислите частное от деления значения арифметического выражения на 14 и
    укажите его в ответе в десятичной системе счисления.
