Задача №14 (вариант 394)
==================================================

394) (А. Богданов) Дано арифметическое выражение:
12x45 + 1x
36 12345
В записи чисел переменной x обозначена неизвестная цифра из допустимого алфавита для
указанных систем счисления. Определите наибольшее значение x, при котором значение данного
арифметического выражения кратно 13. Для найденного значения x вычислите частное от
деления значения арифметического выражения на 13 и укажите его в ответе в десятичной системе
счисления.
