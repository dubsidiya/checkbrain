Задача №14 (вариант 383)
==================================================

383) *(А. Богданов) Операнды арифметического выражения записаны в системе
    счисления с некоторым частично заданным основанием

                              231x324 – 134x222

    В записи чисел переменной x обозначена неизвестная цифра. Определите
    наибольшее значение x, при котором абсолютное значение данного
    арифметического выражения при делении на 50 даёт остаток 31. Для
    найденного значения x вычислите целое частное от деления абсолютного
    значения арифметического выражения на 50 и укажите его в ответе в
    десятичной системе счисления. Основание системы счисления в ответе
    указывать не нужно.
