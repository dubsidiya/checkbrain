Задача №14 (вариант 389)
==================================================

389) *(А. Богданов) Дано арифметическое выражение:
3x15x + 123 + xx + 1x3 + 1x2
15 3x51 1x3 x+1
В записи чисел переменной x обозначена неизвестная цифра. Определите наименьшее значение
x, при котором значение данного арифметического выражения кратно 13. Для найденного
значения x вычислите значение арифметического выражения и укажите его в ответе в системе
счисления с основанием 13. Основание системы счисления в ответе указывать не нужно.
