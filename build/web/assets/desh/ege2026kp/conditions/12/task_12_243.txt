Задача №12 (вариант 243)
==================================================

243)
243) (А. Кабанов) На ленте исполнителя МТ в соседних ячейках записано
    двоичное представление числа 375 без ведущих нулей. Ячейки справа и
    слева от последовательности заполнены пустыми символами «λ». В
    начальный момент времени головка расположена в ближайшей справа от
    последовательности ячейке. Программа для исполнителя:
|      |(     |0     |1     |
|q0    |(, L, |      |      |
|      |q1    |      |      |
|q1    |(, S, |1, L, |0, L, |
|      |q1    |q1    |q1    |


    Команды движения каретки: L – влево, R – вправо, N – отсутствие сдвига,
    S – стоп. Определите десятичное значение числа на ленте после
    выполнения программы.
