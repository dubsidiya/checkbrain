Задача №12 (вариант 205)
==================================================

205)
205) (К. Багдасарян)  На ленте исполнителя МТ в соседних ячейках записана
    последовательность из 600 символов, включающая только нули и единицы.
    Ячейки справа и слева от последовательности заполнены пустыми символами
    «λ». В начальный момент времени головка расположена в ближайшей ячейке
    слева от последовательности. Программа работы исполнителя:
|       |(      |0      |1      |
|q0     |(, R,  |       |       |
|       |q1     |       |       |
|q1     |(, S,  |1, S,  |0, R,  |
|       |q1     |q1     |q1     |


    Команды движения каретки: L – влево, R – вправо, S – стоп. После
    выполнения программы на ленте осталось ровно 250 нулей. Определите
    минимальное возможное число единиц в исходной последовательности.
