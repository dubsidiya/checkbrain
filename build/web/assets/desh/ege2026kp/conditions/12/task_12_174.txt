Задача №12 (вариант 174)
==================================================

174)
174) На ленте исполнителя МТ в соседних ячейках записана последовательность
    из 1000 символов, включающая только нули и единицы. Ячейки справа и
    слева от последовательности заполнены пустыми символами «λ». В
    начальный момент времени головка расположена в ближайшей ячейке слева
    от последовательности. Программа для исполнителя:
|       |(      |0      |1      |
|q0     |(, R,  |       |       |
|       |q1     |       |       |
|q1     |(, S,  |1, R,  |0, R,  |
|       |q1     |q1     |q1     |


    Команды движения каретки: L – влево, R – вправо, S – стоп. После
    выполнения программы на ленте осталось ровно 515 нулей. Определите
    максимально возможное число нулей в исходной последовательности.
