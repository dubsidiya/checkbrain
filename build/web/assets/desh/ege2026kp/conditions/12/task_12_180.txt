Задача №12 (вариант 180)
==================================================

180)
180) (Е. Джобс) На ленте исполнителя МТ в соседних ячейках записана
    последовательность из 1000 символов, включающая только нули, единицы и
    двойки. Ячейки справа и слева от последовательности заполнены пустыми
    символами «λ». В начальный момент времени головка расположена в
    ближайшей ячейке слева от последовательности. Программа для
    исполнителя:
|       |(      |0      |1      |2      |
|q0     |(, R,  |       |       |       |
|       |q1     |       |       |       |
|q1     |(, S,  |1, R,  |2, R,  |0, R,  |
|       |q1     |q1     |q1     |q1     |


    Команды движения каретки: L – влево, R – вправо, S – стоп. Известно,
    что количество символов 0 и 1 в исходной строке было одинаково, а сумма
    значений в исходной строке больше суммы значений в конечной строке на
    200. Какое количество цифр 2 было в исходной строке?
