Задача №12 (вариант 211)
==================================================

211)
211) (Д. Муфаззалов)  На ленте исполнителя МТ в соседних ячейках записана
    последовательность из 100000 символов, включающая только нули и
    единицы. Ячейки справа и слева от последовательности заполнены пустыми
    символами «λ». В начальный момент времени головка расположена в
    ближайшей ячейке справа от последовательности. Программа работы
    исполнителя:
|       |(      |0      |1      |
|q0     |(, L,  |       |       |
|       |q1     |       |       |
|q1     |(, S,  |1, L,  |0, S,  |
|       |q1     |q1     |q1     |


    Команды движения каретки: L – влево, R – вправо, S – стоп. После
    выполнения программы на ленте осталось ровно 343 нуля. Определите
    максимально возможное количество символов последовательности, которые
    могут быть заменены на другой символ в результате выполнения программы.
