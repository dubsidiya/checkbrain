==================================================

169)
169) (М. Фирсов) Исполнитель Счеты преобразует число, записанное на экране.
    У исполнителя есть две команды, которым присвоены номера:
       1. Прибавь 3
       2. Вычти 1
    Первая команда увеличивает число на экране на 3, вторая уменьшает его
    на 1. Программа для исполнителя – это последовательность команд. Будем
    называть циклической программу, при выполнении которой исполнитель на
    каком-то этапе вновь получает начальное число и далее
    поcледовательность команд повторяется. Например, циклической является
    программа следующих преобразований: 1 ( 4 ( 3 ( 2 ( 1 ( 4. Сколько
    существует циклических программ, состоящих из 9 команд?
