117) (С.С. Поляков) Исполнитель Калькулятор преобразует число на экране. У
    исполнителя есть три команды, которым присвоены номера:
       1. Прибавить 1
       2. Прибавить 5
       3. Умножить на 3
    Сколько разных чисел может быть получено из числа 1 с помощью программ,
    состоящих из 7 команд?
