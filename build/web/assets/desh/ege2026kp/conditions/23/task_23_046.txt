46) Исполнитель Июнь15 преобразует число на экране. У исполнителя есть три
    команды, которым присвоены номера:
       1. Прибавить 1
       2. Умножить на 2
       3. Умножить на 3
    Программа для исполнителя Июнь15 – это последовательность команд.
    Сколько существует программ, для которых при исходном числе 2
    результатом является число 28 и при этом траектория вычислений содержит
    число 7?
