Задача №11 (вариант 103)
==================================================

103) (А. Богданов) При регистрации в компьютерной системе каждому файлу
    присваивается идентификатор фиксированной длины из набора символов,
    включающего десятичные цифры, а также 16 заглавных латинских букв.
    Каждый символ кодируется с помощью одинакового и минимального
    количества бит. Для хранения идентификатора выделяется минимально
    возможное количество байт. Какое максимальное количество различных
    идентификаторов можно создать, если для хранения 1000 идентификаторов
    выделяется 3 килобайта памяти?
