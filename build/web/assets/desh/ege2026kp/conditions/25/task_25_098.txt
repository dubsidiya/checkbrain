Задача №25 (вариант 98)
==================================================

98) (Е. Джобс) Среди целых чисел, принадлежащих числовому отрезку [228224;
    531135], найдите числа, среди делителей которых есть хотя бы 4
    различных куба натуральных нечетных чисел. Для каждого найденного числа
    запишите в ответе само число, количество таких делителей и наибольший
    из них. В качестве делителей не рассматривать число 1 и само
    исследуемое число. Так, например, для числа 8 учитываются только
    делители 2 и 4.
