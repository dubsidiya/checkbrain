Задача №25 (вариант 158)
==================================================

158) (Е. Джобс) Найдите возрастающую последовательность из 5 чисел,
    начинающуюся с 700000, такую, что каждый следующий элемент – это
    минимальное число, количество делителей которого превосходит количество
    делителей предыдущего числа. Для каждого элемента последовательности
    запишите сначала само число, а затем количество его натуральных
    делителей.
