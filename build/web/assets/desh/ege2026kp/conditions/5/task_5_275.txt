Задача №5 (вариант 275)
==================================================

275)
275) На вход алгоритма подаётся натуральное число N. Алгоритм строит по
    нему новое число R следующим образом.
    1. Вычисляется сумма S1 всех чётных цифр десятичной записи числа N.
    Если чётных цифр нет, сумма S1 считается равной 0.
    2. Вычисляется сумма S2 всех цифр десятичной записи числа N, стоящих на
    позициях с чётными номерами. Позиции нумеруются слева направо, начиная
    с 1. Для однозначных чисел сумма S2 считается равной 0.
    3. Вычисляется результат R как модуль разности S1 и S2.
    Пример: Дано число N = 4321. Сумма чётных цифр S1 = 4 + 2 = 6. Сумма
    цифр в позициях с чётными номерами S2 = 3 + 1 = 4. Результат работы
    алгоритма R = 6 – 4 = 2.
    Укажите наименьшее число, в результате обработки которого по данному
    алгоритму получится число 29.
