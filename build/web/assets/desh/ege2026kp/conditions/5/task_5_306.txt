Задача №5 (вариант 306)
==================================================

306)
306) (А. Игнатюк) Компьютер по имени Иннокентий преобразует натуральное
    число N по следующим правилам и получает число R:
        1) Строится двоичная запись числа N.
        2) Если количество цифр в двоичной записи числа N четно, то справа
          приписывается 10, если нечётно, то слева приписывается 11.
    Полученная таким образом запись является двоичной записью искомого
    числа R. Найдите количество чисел N из отрезка [100; 200], для которых
    результат работы компьютера будет четным.
