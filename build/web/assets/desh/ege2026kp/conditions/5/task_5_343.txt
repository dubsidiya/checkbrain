Задача №5 (вариант 343)
==================================================

343)
343) (Н. Сафронов) Алгоритм получает на вход натуральное число N и строит
    по нему новое число R следующим образом:
        1. Строится троичная запись числа N.
        2. К этой записи дописываются разряды по следующему правилу. Если
         сумма троичных разрядов кратна 3, слева дописывается 20, иначе
         слева дописывается 10.
        3. Полученная таким образом запись является троичной записью
         искомого числа R.
    Например, для числа 10 троичная запись 1013 преобразуется в запись
    101013 = 91, для числа 11 троичная запись 1023 преобразуется в 201023 =
    173.
    Укажите максимальное значение N, после обработки которого с помощью
    этого алгоритма получается число R, меньшее чем 100.
