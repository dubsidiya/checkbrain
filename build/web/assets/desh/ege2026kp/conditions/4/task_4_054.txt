Задача №4 (вариант 54)
==================================================

54)
54) (ЕГЭ-2024) По каналу связи передаются сообщения, содержащие только
    буквы из набора: А, Т, К, С, Н. Для передачи используется двоичный код,
    удовлетворяющий условию Фано. Кодовые слова для некоторых букв
    известны: Н – 11, С – 101. Для трёх оставшихся букв К, Т и А кодовые
    слова неизвестны. Какое количество двоичных знаков требуется для
    кодирования слова КАСАТКА, если известно, что оно закодировано
    минимально возможным количеством двоичных знаков?
