Задача №4 (вариант 31)
==================================================

31)
31)  (Е. Джобс) По каналу связи передаются сообщения, содержащие только
    шесть букв: Я, Н, В, А, Р, Ь. Для передачи используется двоичный код,
    удовлетворяющий условию Фано. Кодовые слова для некоторых букв
    известны: Н – 00, В – 01, Р – 10, Ь – 111. Укажите минимально возможную
    длину закодированной последовательности для слова ВАРВАР.
