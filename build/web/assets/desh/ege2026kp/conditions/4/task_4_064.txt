Задача №4 (вариант 64)
==================================================

64)
64) (О. Кувватова) По каналу связи передаются сообщения, содержащие только
    буквы из набора: А, В, Л, Н, З, И, Ы, М. Для передачи используется
    двоичный код, удовлетворяющий условию Фано. Кодовые слова для некоторых
    букв известны: Л – 01, Ы – 1111. Для оставшихся букв данного алфавита
    кодовые слова неизвестны. Какое количество двоичных знаков требуется
    для кодирования слова ИЗВИЛИНЫ, если известно, что оно закодировано
    минимально возможным количеством двоичных знаков?
