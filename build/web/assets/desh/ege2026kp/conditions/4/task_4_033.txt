Задача №4 (вариант 33)
==================================================

33)
33)  (Е. Джобс) По каналу связи передаются сообщения, содержащие только
    шесть букв: Д, Е, Н, Ь, Г, И. Для передачи используется двоичный код,
    удовлетворяющий условию Фано. Кодовые слова для некоторых букв
    известны: Д – 101, Г – 01, Е – 001. Какова минимально возможная длина
    закодированной последовательности для слова ГИГИЕН?
