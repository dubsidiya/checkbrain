# Извлечение задач из Word файлов

## Установка зависимостей

```bash
# Установите python-docx для работы с .docx файлами
pip3 install python-docx

# Установите antiword для работы со старыми .doc файлами (macOS)
brew install antiword

# Или для Linux
sudo apt-get install antiword
```

## Запуск скрипта

```bash
cd /Users/vladkharin/StudioProjects/checkbrain
python3 extract_tasks.py
```

## Что делает скрипт

1. Находит все Word файлы (ege*.doc, ege*.docx) в папке `desh/ege2026kp/`
2. Извлекает текст из каждого файла
3. Разбивает текст на отдельные задачи (если возможно)
4. Сохраняет каждую задачу в отдельный файл в структуре:
   ```
   desh/ege2026kp/
     conditions/
       1/
         task_1_001.txt
         task_1_002.txt
         ...
       2/
         task_2_001.txt
         ...
   ```

## Структура файлов задач

Каждый файл содержит:
- Заголовок с номером задачи и вариантом
- Полное условие задачи

## Использование в приложении

После извлечения задач:
1. Откройте приложение
2. Перейдите на вкладку "Kompege"
3. Выберите номера задач (1-27)
4. Для каждого номера будет выбрана случайная задача из доступных
5. Нажмите "Начать вариант"

## Примечания

- Если файл содержит несколько задач, они будут сохранены отдельно
- Если разбиение не удалось, весь файл сохраняется как одна задача
- Для улучшения разбиения можно вручную отредактировать скрипт `split_into_tasks()`
