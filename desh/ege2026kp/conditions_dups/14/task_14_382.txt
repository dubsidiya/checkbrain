Задача №14 (вариант 382)
==================================================

382)
392) (И. Карпачев) Дано арифметическое выражение:

                              3B8x117 + 2x9x317

    В записи чисел переменной x обозначена неизвестная цифра из алфавита 17-
    ричной системы счисления. Найдите наименьшее значение переменной x, при
    котором запись значения этого выражения в 6-ричной системе счисления
    содержит ровно три цифры «5». В ответе для найденного значения x
    укажите значение выражения в десятичной системе счисления. Основание
    системы счисления в ответе указывать не нужно.
