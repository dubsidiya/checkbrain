Задача №12 (вариант 261)
==================================================

261)
261) На ленте исполнителя МТ в соседних ячейках записано двоичное
    представление целого положительного числа без ведущих нулей. Ячейки
    справа и слева от последовательности заполнены пустыми символами «λ». В
    начальный момент времени головка расположена в ближайшей слева от
    последовательности ячейке. Программа для исполнителя:
|      |(     |0     |1     |
|q0    |(, R, |      |      |
|      |q1    |      |      |
|q1    |(, S, |1, R, |0, R, |
|      |q1    |q1    |q1    |


    Команды движения каретки: L – влево, R – вправо, N – отсутствие сдвига,
    S – стоп. После выполнения программы на ленте оказалось двоичная запись
    числа 415. Определите десятичное значение наибольшего числа, меньшего,
    чем 70000, которое могло быть записано на ленте до начала работы
    программы.
