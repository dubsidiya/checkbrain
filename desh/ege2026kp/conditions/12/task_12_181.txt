Задача №12 (вариант 181)
==================================================

181)
181) (Е. Джобс) На ленте исполнителя МТ в соседних ячейках записана
    последовательность из 1000 символов, включающая только нули, единицы и
    двойки. Ячейки справа и слева от последовательности заполнены пустыми
    символами «λ». В начальный момент времени головка расположена в
    ближайшей ячейке слева от последовательности. Программа для
    исполнителя:
|       |(      |0      |1      |2      |
|q0     |(, R,  |       |       |       |
|       |q1     |       |       |       |
|q1     |(, S,  |1, R,  |2, R,  |0, R,  |
|       |q1     |q1     |q1     |q1     |


    Команды движения каретки: L – влево, R – вправо, S – стоп. Известно,
    что каждый из символов 0, 1 и 2 есть в исходной строке. Суммы значений
    в начальной и конечной строках кратны 5, при этом больше 0. При каком
    максимальном количестве символов 0 в исходной строке это возможно?
