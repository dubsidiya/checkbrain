Задача №23 (вариант 159)
==================================================

159)
159) (Б. Михлин) Исполнитель К22 преобразует число, записанное на экране. У
    исполнителя есть три команды, которым присвоены номера:
       1. Прибавь 1
       2. Прибавь 4
       3. Прибавь меньшее число Фибоначчи
    Первая из них увеличивает число на экране на 1, вторая увеличивает
    число на 4.
    Третья команда увеличивает число на ближайшее число из ряда Фибоначчи
    меньшее,
    чем число на экране (например, для числа 3 будет получено 5 = 3 + 2, а
    для числа 7 будет получено 12 = 7 + 5). Программа для исполнителя – это
    последовательность команд.
    Сколько существует программ, которые преобразуют исходное число 2 в
    число 16?
    Примечание. Числа Фибоначчи – это ряд чисел, в котором первое и второе
    число равны единице, а каждое следующее число равно сумме двух
    предыдущих чисел ряда: 1, 1, 2, 3, 5, 8, 13, ...
