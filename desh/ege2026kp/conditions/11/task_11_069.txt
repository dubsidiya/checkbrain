Задача №11 (вариант 69)
==================================================

69) (А. Жуков) Каждый сотрудник предприятия получает электронный пропуск,
    на котором записаны его личный код, номер кабинета, в котором он
    работает и некоторая дополнительная информация. Личный код сотрудника
    состоит из 5 символов, каждый из которых может быть одной из 23-х
    прописных латинских букв (не используются буквы B, О и I) или
    десятичной цифрой (от 0 до 9). Для записи кода на пропуске отведено
    минимально возможное целое число байт. При этом используют посимвольное
    кодирование, все символы кодируют одинаковым минимально возможным
    количеством бит. При нумерации кабинетов используют следующую систему –
    первая цифра номера обозначает этаж, оставшиеся две – номер кабинета на
    этаже, при этом используются только целые числа из диапазона от 100 до
    299 включительно. Для записи номера кабинета на пропуске используется
    минимальное и целое число байт. Всего на пропуске хранится 56 байт
    данных. Сколько байт выделено для хранения дополнительных сведений об
    одном сотруднике? В ответе запишите только целое число – количество
    байт.
