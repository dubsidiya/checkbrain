Задача №11 (вариант 87)
==================================================

87)
87) (А. Богданов) При регистрации в компьютерной системе на каждого
    сотрудника заводится запись из двух полей: идентификатор и биометрия
    лица. Идентификатор состоит из семи эмодзи, каждый из которых
    закодирован в виде трех шестнадцатеричных цифр из диапазона U+1Fххх
    Unicode символов. Биометрия лица представлена десятью десятичными
    трехзначными числами.  В базе данных для хранения каждого поля отведено
    минимально возможное целое число байт. При этом используют посимвольное
    кодирование полей. Все символы (или числа) поля кодируют одинаковым и
    минимально возможным количеством бит. Определите объём памяти (в
    Кбайт), необходимый для хранения 256 записей. В ответе запишите только
    целое число – количество Кбайт.
