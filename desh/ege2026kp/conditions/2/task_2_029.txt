29) Дан фрагмент таблицы истинности выражения F (см. таблицу справа). Какое X Y Z F
выражение соответствует F? 1 0 1 0
1) (X ~ Z)  (¬X → Y) 2) (¬X ~ Z)  (¬X → Y) 0 1 0 1
3) (X ~ ¬Z)  (¬X → Y) 4) (X ~ Z)  ¬(Y → Z) 1 1 1 0
Знак ~ означает «эквивалентность», то есть «X ~ Z» значит «значения X и Z совпадают».
