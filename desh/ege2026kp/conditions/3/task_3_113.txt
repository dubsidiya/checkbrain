Задача №3 (вариант 113)
==================================================

113) (Е. Морох) Используя информацию из базы данных в файле 3-107.xls (см.
    схему данных в разборе задачи 107), определите клубы, которые побеждали
    в чемпионате страны, но ни разу не выиграли национальный кубок. Из этих
    клубов выберите клуб с наибольшим количеством побед в чемпионате
    страны. В ответе укажите ID  страны, которую он представляет. Если
    таких клубов несколько, выберите наибольший ID страны.
