==================================================

89)
89) Алгоритм вычисления функции F(n), где n – натуральное число, задан
    следующими соотношениями:
         F(n) = 1,  при n < 2,
         F(n) = F(n / 3) – 1, когда n ( 2 и делится на 3,
         F(n) = F(n – 1) + 17 , когда n ( 2 и не делится на 3.

    Назовите количество значений n на отрезке [1;100000], для которых F(n)
    равно 43.
