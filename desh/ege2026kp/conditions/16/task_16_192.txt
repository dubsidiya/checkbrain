Задача №16 (вариант 192)
==================================================

192)
192) (Д. Паршиков) Алгоритм вычисления функции F(n), где n – натуральное
    число, задан следующими соотношениями:
         F(n) = n, если n ≥ 3000,
         F(n) = n + 2x  + F(n + 2), если n < 3000.
    При каком целом значении x выполняется равенство F(28) –  F(34) = 324?
