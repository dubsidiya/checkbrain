Задача №16 (вариант 99)
==================================================

99)
99) (А. Богданов) Алгоритм вычисления функции F(n), где n – целое
    неотрицательное число, задан следующими соотношениями:
         F(0) = 0,
         F(n) = 1, когда 0 < n < 3,
         F(n) = F(n – 2) + F(n – 1) , когда n ≥ 3.
