==================================================

85)
85) Алгоритм вычисления функции F(n), где n – натуральное число, задан
    следующими соотношениями:
         F(1) = 1,
         F(n) = F(n / 2) + 1, когда n ( 2 и чётное,
         F(n) = F(n – 1) + n , когда n ( 2 и нечётное.

    Назовите количество значений n на отрезке [1;100000], для которых F(n)
    равно 16.
