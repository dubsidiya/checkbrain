Задача №16 (вариант 100)
==================================================

100)
100) (Е. Джобс) Алгоритм вычисления функции F(n), где n – целое
    неотрицательное число, задан следующими соотношениями:
         F(n) = n + 3, при n ≤ 3
         F(n) = F(n – 2) + n, при n > 3 и четном значении F(n-1),
         F(n) = F(n – 2) + 2· n, при n > 3 и нечетном значении F(n-1)
