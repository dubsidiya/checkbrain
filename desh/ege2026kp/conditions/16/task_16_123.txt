Задача №16 (вариант 123)
==================================================

123)
123) Алгоритм вычисления функции F(n), где n – целое неотрицательное число,
    задан следующими соотношениями:
         F(0) = 8
         F(n) = 5 + F(n / 3), если n > 0 и n делится на 3,
         F(n) = F(n // 3) в остальных случаях.
    Здесь // означает деление нацело. Определите количество значений n на
    отрезке [1, 100 000 000], для которых F(n) = 18.
