Задача №7 (вариант 107)
==================================================

107) Для хранения сжатого произвольного растрового изображения размером
    1200x1600 пикселей отведено 1850 Кбайт памяти без учёта размера
    заголовка файла. Для кодирования цвета каждого пикселя используется
    одинаковое количество бит, коды пикселей записываются в файл один за
    другим без промежутков. При сохранении данные сжимаются, размер
    итогового файла после сжатия становится на 20% меньше исходного. Какое
    максимальное количество цветов можно использовать в изображении?
