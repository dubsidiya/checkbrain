==================================================

210) (Е. Джобс) Исполнитель преобразует двузначное число, записанное на экране. У исполнителя
есть две команды, которым присвоены номера:
31 http://kpolyakov. spb .ru
© К. Поляков, 2009-2025
1. Сложи разряды числа
2. Перемножь разряды числа
Выполняя первую из них, исполнитель складывает разряды числа и выводит соответствующее
значение на экран. При выполнении второй команды находится произведение разрядов, которое
выводится на экран. Программой для исполнителя называется последовательность команд.
Например, программа 221 примененная к числу 93 выполнится следующим образом: 9*3 = 27, 2*7
= 14, 1+4 = 5. Найдите количество различных двузначных чисел, которые этот исполнитель может
преобразовать в число 8?
