Задача №15 (вариант 519)
==================================================

519) (А. Богданов) На числовой прямой даны два отрезка: B = [23; 37] и C = [41; 73]. Укажите
наименьшую длину такого отрезка А, для которого логическое выражение
¬((¬(x ∈ B) → (x ∈ C)) → (x ∈ A))
тождественно ложно, т. е. принимает значение 0 при любом значении переменной x.
