==================================================

111) Дан фрагмент таблицы истинности для выражения F:
x1 x2 x3 x4 x5 x6 x7 F
0 1 1
0 0 0
0 1 0
Каким выражением может быть F?
1) x1  (x2  x3)  ¬x4  x5  x6  ¬x7
2) x1  (¬x2  x3)  ¬x4  ¬x5  x6  ¬x7
3) ¬x1  (x2  ¬x3)  x4  ¬x5  x6  x7
4) x1  (x2  ¬x3)  ¬x4  x5  ¬x6  x7
