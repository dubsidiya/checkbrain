Задача №11 (вариант 159)
==================================================

159) (О. Лысенков) Петя кодирует сообщение, состоящее из русских заглавных и строчных букв,
десятичных цифр, а также смайликов. Всего в алфавите имеется 50 смайликов, причем каждый из
них может быть одного из 5 цветов (светлый, смуглый, желтоватый, коричневый и темный). Для
кодирования каждого символа используется минимальное целое число бит, одинаковое для всех
символов. Сколько байтов нужно выделить для хранения сообщения, содержащего 1016
символов?
