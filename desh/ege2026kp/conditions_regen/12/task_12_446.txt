Задача №12 (вариант 446)
==================================================

446) *(Г. Гутман) На ленте исполнителя МТ в соседних ячейках записана последовательность из 2025
символов, включающая только нули и единицы. Ячейки справа и слева от последовательности
заполнены пустыми символами «λ». В начальный момент времени головка расположена в
ближайшей ячейке слева от последовательности. Программа для исполнителя:
 0 1
q , R, q 0, N, q 1, N, q
0 0 1 1
q , S, q 0, R, q 1, R, q
1 0 1 2
q , S, q 0, R, q 1, R, q
2 0 1 3
q , S, q 0, R, q 0, R, q
3 0 3 1
Команды движения каретки: L – влево, R – вправо, N – отсутствие сдвига, S – стоп. После
выполнения программы преобразованная строка совпала с исходной строкой. Определите
максимально возможное число единиц в исходной последовательности.
