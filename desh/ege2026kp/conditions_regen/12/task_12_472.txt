Задача №12 (вариант 472)
==================================================

472) (А. Кабанов) На ленте исполнителя МТ в соседних ячейках записана последовательность из 221
единиц. Ячейки справа и слева от последовательности заполнены пустыми символами «λ». В
начальный момент времени головка расположена в ближайшей справа от последовательности
ячейке. Программа для исполнителя:
 0 1
q , L, q
0 1
q 0, L, q 1, L, q
1 3 3
q , S, q 1, L, q 0, L, q
2 2 3 1
q , S, q 0, L, q 1, L, q
3 3 2 2
Команды движения каретки: L – влево, R – вправо, N – отсутствие сдвига, S – стоп. Определите
число единиц на ленте после выполнения этой программы.
