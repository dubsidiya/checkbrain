Задача №12 (вариант 478)
==================================================

478) (А. Кабанов) На ленте исполнителя МТ в соседних ячейках записано двоичное представление
числа 992 без ведущих нулей. Ячейки справа и слева от последовательности заполнены пустыми
символами «λ». В начальный момент времени головка расположена в ближайшей справа от
последовательности ячейке. Программа для исполнителя:
 0 1
q , L, q
0 1
q 1, L, q 0, L, q
1 2 2
q , S, q 1, L, q 1, L, q
2 2 2 3
q , S, q 0, L, q 0, L, q
3 3 3 2
Команды движения каретки: L – влево, R – вправо, N – отсутствие сдвига, S – стоп. Определите
десятичное значение числа на ленте после выполнения этой программы.
