Задача №12 (вариант 456)
==================================================

456) *(Г. Гутман) На ленте исполнителя МТ в соседних ячейках записана последовательность из 780
символов, включающая только нули и единицы. Ячейки справа и слева от последовательности
заполнены пустыми символами «λ». В начальный момент времени головка расположена в
ближайшей ячейке слева от последовательности. Программа для исполнителя:
 0 1
q , R, q
0 1
q , S, q 0, R, q 1, R, q
1 1 2 2
q , S, q 1, R, q 0, R, q
2 2 3 3
q , S, q 1, R, q 0, R, q
3 3 1 1
Команды движения каретки: L – влево, R – вправо, N – отсутствие сдвига, S – стоп. После
выполнения программы в полученной последовательности оказалось поровну символов 0 и 1.
Определите минимально возможное число нулей в исходной последовательности.
