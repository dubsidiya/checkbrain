Задача №22 (вариант 147)
==================================================

147) (Р. Косов) В файле 22-145.ods содержится информация о совокупности N вычислительных
процессов, которые могут выполняться параллельно или последовательно… (Условие совпадает с
условием задачи из демо-варианта 2024 года). Определите количество процессов, выполнение
которых начнётся после завершения процесса 107, если каждый процесс начинается в самое
раннее допустимое время.

Например, для приведённой таблицы найдём количество процессов, выполнение которых
начинается после завершения процесса 2. Минимальное время, которое требуется для
завершения процесса 2, равно 7 мс. После этого стартует только один процесс – 3.
