Задача №18 (вариант 63)
==================================================

63) (В.Н. Шубинкин) Исходные данные для Робота (см. задачу Р-00) записаны в файле 18-2.xls в
виде электронной таблицы прямоугольной формы. Робот может двигаться только вниз и вправо.
Робот может брать монеты только с тех клеток, где количество монет нечётно. Если количество
монет чётно, то Робот не берёт в этой клетке ни одной монеты. Определите максимальную и
минимальную денежную сумму, которую может собрать Робот, пройдя из левой верхней клетки в
правую нижнюю. В ответе укажите два числа – сначала максимальную сумму, затем
минимальную.
