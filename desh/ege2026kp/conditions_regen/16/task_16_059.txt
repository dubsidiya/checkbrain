Задача №16 (вариант 59)
==================================================

59) Алгоритм вычисления функции F(n) задан следующими соотношениями:
F(n) = n · n + 5 · n + 4, при n > 30
F(n) = F(n+1) + 3 · F(n+4), при чётных n  30
F(n) = 2 · F(n+2) + F(n+5), при нечётных n  30
Определите количество натуральных значений n из отрезка [1; 1000], для которых сумма цифр
значения F(n) равна 27.
