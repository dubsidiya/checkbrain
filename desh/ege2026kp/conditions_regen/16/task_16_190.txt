Задача №16 (вариант 190)
==================================================

190) (Н. Сафронов) Алгоритм вычисления значения функции F(n), где n – натуральное число, задан
следующими соотношениями:
F(n) = 1, если n = 1,
F(n) = n + F(n – 1), если n > 1.
Определите количество значений n на отрезке [1, 100], для которых значение выражения
F(2023) // F ( n ) будет четным. Здесь // обозначает целочисленное деление.
