Задача №16 (вариант 211)
==================================================

211) *(А. Минак) Алгоритм вычисления значения функции F(n), где n – натуральное число, задан
следующими соотношениями:
F(n) = 2, при n > 2024;
F(n) = 1, при n = 2024;
F(n) = n · (n + 1) + F(n + 1) – F(n + 2), если n < 2024.
Чему равно значение выражения F(100) – F(10) + F(2020)?
