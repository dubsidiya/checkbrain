Задача №16 (вариант 60)
==================================================

60) Алгоритм вычисления функции F(n) задан следующими соотношениями:
F(n) = n · n + 4 · n + 3, при n > 25
F(n) = F(n+1) + 2 · F(n+4), при n  25, кратных 3
F(n) = F(n+2) + 3 · F(n+5), при n  25, не кратных 3
Определите количество натуральных значений n из отрезка [1; 1000], для которых сумма цифр
значения F(n) равна 24.
