Задача №16 (вариант 98)
==================================================

98) Алгоритм вычисления функции F(n), где n – целое неотрицательное число, задан следующими
соотношениями:
F(0) = 0,
F(n) = F(n / 2) + 3, когда n > 0 и делится на 2,
F(n) = 2·F(n – 1) + 1 , когда n > 0 и не делится на 2.
Сколько различных значений может принимать функция F(n) при n, принадлежащих отрезку [1;
1000]?
