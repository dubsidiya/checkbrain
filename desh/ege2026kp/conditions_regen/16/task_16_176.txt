Задача №16 (вариант 176)
==================================================

176) Алгоритм вычисления функций F(n) и G(n), где n – натуральное число, задан следующими
соотношениями:
F(n) = n, если n > 1000000;
F(n) = n + F(4n), если n ≤ 1000000.
G(n) = F(n) / n.
Определите количество натуральных чисел n (включая n = 2000), для которых G(n) = G(2000).
