193) Алгоритм вычисления функции F(n), где n – натуральное число, задан следующими
соотношениями:
F(n) = n, если n ≥ 1300,
F(n) = n · F(n + 1), если n < 1300 и n – нечётное;
F(n) = n · F(n + 2) / 4, если n < 1300 и n – чётное.
Чему равно значение выражения F(1286) / F(1290)?
