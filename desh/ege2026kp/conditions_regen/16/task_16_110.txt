Задача №16 (вариант 110)
==================================================

110) Алгоритм вычисления функции F(n), где n – целое неотрицательное число, задан следующими
соотношениями:
F(n) = 0 при n = 0
F(n) = F(n/2) – 1 при n > 0 для чётных n
F(n) = 1 + F(n–1) при n > 0 для нечётных n
Сколько существует чисел n, меньших 1000, для которых значение F(n) будет равно 0?
