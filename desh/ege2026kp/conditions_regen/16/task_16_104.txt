Задача №16 (вариант 104)
==================================================

104) (П. Волгин) Алгоритм вычисления функции F(n), где n – целое неотрицательное число, задан
следующими соотношениями:
F(0) = 1
F(n) = F(n–1), при 0 < n ≤ 10
F(n) = 2,2*F(n–3), при 10 < n <100
F(n) = 1,7*F(n–2), при n ≥ 100
Чему равна сумма цифр целой части F(40)?
