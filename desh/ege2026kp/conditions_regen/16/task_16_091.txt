Задача №16 (вариант 91)
==================================================

91) Алгоритм вычисления функции F(n), где n – натуральное число, задан следующими
соотношениями:
F(n) = 1, при n < 2,
F(n) = F(n / 2) + 1, когда n  2 и чётное,
F(n) = F(n – 3) + 3 , когда n  2 и нечётное.
Назовите минимальное значение n, для которого F(n) равно 31.
