Задача №16 (вариант 49)
==================================================

49) Алгоритм вычисления функции F(n) задан следующими соотношениями:
F(n) = n при n ≤ 3;
F(n) = 2 · n + F(n – 1) при чётных n > 3;
F(n) = n · n + F(n – 2) при нечётных n > 3;
Определите количество натуральных значений n из отрезка [1; 100], при которых значение F(n)
кратно 3.
