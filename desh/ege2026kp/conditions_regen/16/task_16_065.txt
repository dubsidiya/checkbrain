Задача №16 (вариант 65)
==================================================

65) Алгоритм вычисления функции F(n) задан следующими соотношениями:
F(n) = n + 15, при n  5
F(n) = F(n // 2) + n · n · n – 1, при чётных n > 5
F(n) = F(n–1) + 2 · n · n + 1, при нечётных n > 5
Здесь «//» обозначает деление нацело. Определите количество натуральных значений n из
отрезка [1; 1000], для которых значения F(n) содержит не менее двух цифр 8.
