Задача №16 (вариант 180)
==================================================

180) (А. Богданов) Обозначим частное от деления натурального числа a на натуральное число b как
a // b, а остаток как a%b. Алгоритм вычисления функции F(n), где n – натуральное число, задан
следующими соотношениями:
F(n) = n // 3 + n % 3, если n < 9;
F(n) = F(n // 9) + F(n % 9), если n ≥ 9.
Определите количество значений n < 99, для которых функция F(n) = 33.
