Задача №16 (вариант 97)
==================================================

97) Алгоритм вычисления функции F(n), где n – целое неотрицательное число, задан следующими
соотношениями:
F(0) = 0,
F(n) = F(n / 2), когда n > 0 и делится на 2,
F(n) = F(n – 1) + 3 , когда n > 0 и не делится на 2.
Сколько существует значений n, принадлежащих отрезку [1; 1000], для которых F(n) равно 18?
