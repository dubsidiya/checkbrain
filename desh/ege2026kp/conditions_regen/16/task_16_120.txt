Задача №16 (вариант 120)
==================================================

120) Алгоритм вычисления функции F(n), где n – целое неотрицательное число, задан следующими
соотношениями:
F(0) = 5
F(n) = 1 + F(n / 2), если n > 0 и n чётное,
F(n) = F(n // 2) в остальных случаях.
Здесь // означает деление нацело. Определите количество значений n на отрезке [1, 1 000 000
000], для которых F(n) = 7.
