Задача №16 (вариант 114)
==================================================

114) (П. Волгин) Алгоритм вычисления функции F(n), где n – целое неотрицательное число, задан
следующими соотношениями:
F(n) = 1 при n = 0
F(n) = 7·(n – 1) + F(n–1) при n > 0
Сколько существует значений n на отрезке [2, 200], для которых значение функции F(n) является
простым числом?
