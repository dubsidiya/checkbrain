Задача №16 (вариант 178)
==================================================

178) Алгоритм вычисления функций F(n) и G(n), где n – натуральное число, задан следующими
соотношениями:
F(n) = n, если n > 2000000;
F(n) = 7n + F(3n), если n ≤ 2000000.
G(n) = F(n) / n.
Определите количество натуральных чисел n (включая n = 12345), для которых G(n) = G(12345).
