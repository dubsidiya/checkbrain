223) (Д. Муфаззалов) Обозначим через a // b целую часть от частного при делении числа a на число b.

соотношениями:
F(n) = 1 при n ≤ 2025;
F(n) = F( (n + 2024) // 2025) + 1, если n > 2025.
Чему равно значение F(11 + 22 + 33 + … + 20252025)?
