Задача №16 (вариант 189)
==================================================

189) (А. Богданов) Обозначим операцию целочисленного деления с округлением вниз как «//», а
нахождения остатка деления через «%». Например, 8 // 3 == 2 и 7 % 3 == 1. Алгоритм вычисления
функции F(n), где n – натуральное число, задан следующими соотношениями:
F(n) = n, если n < 2,
F(n) = F(n // 2) + F(n % 2), если n ≥ 2.
Определите количество натуральных чисел, меньших 230, для которых F(n) = 27?
