Задача №4 (вариант 263)
==================================================

263) (И. Баженов) По каналу связи передаются сообщения, содержащие только буквы из набора: Р, Е,
Ф, И, Ж, А, Т, О. Для передачи используется двоичный код, удовлетворяющий условию Фано.
Кодовые слова для некоторых букв известны: И – 101, Ф – 1100. Для шести оставшихся букв
кодовые слова неизвестны. Какое количество двоичных знаков потребуется для кодирования
слова РЕФРИЖЕРАТОР, если известно, что оно закодировано минимально возможным
количеством двоичных знаков?
