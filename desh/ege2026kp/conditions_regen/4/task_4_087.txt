Задача №4 (вариант 87)
==================================================

87) По каналу связи передаются сообщения, содержащие только 4 буквы М, А, Р, Т; для передачи
используется двоичный код, допускающий однозначное декодирование. Для букв М, А, Р
используются такие кодовые слова: М: 010, А: 1, Р: 011.
Укажите кратчайшее кодовое слово для буквы Т, при котором код будет допускать однозначное
декодирование. Если таких кодов несколько, укажите код с наименьшим числовым значением.
