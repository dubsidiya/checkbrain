Задача №4 (вариант 200)
==================================================

200) (А. Богданов) По каналу связи передаются шифрованные сообщения, содержащие только 10
букв. Для передачи используется неравномерный двоичный код. Для девяти букв слова известны:
{ 11; 0011; 101; 100; 0010; 0101; 0001; 0000; 011 }
Укажите кратчайшее кодовое слово для десятой буквы, при котором код будет удовлетворять
условию Фано. Если таких кодов несколько, укажите код с наибольшим числовым значением.
