Задача №4 (вариант 205)
==================================================

205) (А. Минак) Алфавит передаваемых по каналу связи информационных сообщений содержит
только восемь букв: A, E, G, H, N, R, S, T. Для передачи используется двоичный код,
удовлетворяющий условию однозначного декодирования. Кодовые слова некоторых букв
известны: R – 0, N – 011, S – 1111. Какое наименьшее количество двоичных разрядов потребуется
для кодирования слова STRENGTHS?
