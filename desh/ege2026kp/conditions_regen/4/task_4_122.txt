Задача №4 (вариант 122)
==================================================

122) По каналу связи передаются сообщения, содержащие только пять букв: Р, А, Н, Е, Т. Для
передачи используется двоичный код, удовлетворяющий условию Фано. Для буквы А
используется кодовое слово 0; для буквы Е используется кодовое слово 10. Какова минимальная
общая длина кодовых слов для всех пяти букв?
