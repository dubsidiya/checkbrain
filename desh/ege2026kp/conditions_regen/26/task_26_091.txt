Задача №26 (вариант 91)
==================================================

91) (М. Шагитов) На склад торговой базы поступают товары в ящиках, которые имеют стандартный
размер и разный вес. Ящики размещаются в контейнерах, каждый из которых вмещает два пакета
с суммарным весом не более D кг. Если при этом какие-то пакеты не удалось упаковать в
контейнеры парами (из-за слишком большого веса), они размещаются по одному. Гарантируется,
что вес каждого ящика не превышает D. Определите наибольшее количество контейнеров, в
которые можно поместить по 2 ящика, и минимально возможный суммарный вес ящиков,
которые размещаются по одному в контейнере.

Входные данные представлены в файле 26-91.txt следующим образом. В первой строке
записаны два числа, разделённые пробелом: N – количество товаров (1 ≤ N ≤ 5000) и D –
суммарный допустимый вес товаров в контейнере (1 ≤ D ≤ 1000). В каждой из следующих N строк
записано одно натуральное число: вес ящика, поступившего на склад.
Запишите в ответе два целых числа: сначала наибольшее количество контейнеров, в которые
можно поместить по 2 ящика, затем минимально возможный суммарный вес ящиков, которые
размещаются по одному в контейнере.

и {54, 52}, а товары {81} и {83} (общим весом 164 кг) помещаются в отдельных контейнерах. Ответ:
4 164.
