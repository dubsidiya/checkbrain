В записи чисел переменной x обозначена неизвестная цифра из алфавита 17-ричной системы
счисления. Найдите наименьшее значение переменной x, при котором запись значения этого
выражения в 6-ричной системе счисления содержит ровно три цифры «5». В ответе для
найденного значения x укажите значение выражения в десятичной системе счисления. Основание
системы счисления в ответе указывать не нужно.
