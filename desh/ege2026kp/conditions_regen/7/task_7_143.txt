Задача №7 (вариант 143)
==================================================

143) (Е. Пеньков) Для хранения произвольного растрового изображения размером 1280 × 720
пикселей отведено 1575 Кбайт памяти без учёта размера заголовка файла. При кодировании
цвета каждого пикселя используется одинаковое количество бит, кратное 7: для каждых пяти бит
цвета дописываются два дополнительных бита контроля чётности для повышения надёжности
передачи данных. Коды пикселей записываются в файл один за другим без промежутков. Какое
максимальное количество цветов можно использовать в изображении?
